# Calculadora Shopee

Web app estático para calcular preços de anúncios e valores líquidos de pedidos Shopee.

## Uso

Abra `index.html` em qualquer navegador moderno. Não é necessário instalar dependências, criar conta ou ter conexão com a internet.

As configurações da taxa por pedido ficam salvas somente no navegador do aparelho.

## Testes

Com Node.js instalado, execute:

```powershell
node --test tests/calculator.test.js
```
